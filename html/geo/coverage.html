This is a live map from the galmon.eu project. This map can plot if a fix is possible ('Coverage') or if the positioning, horizontal or vertical precision will be bad (PDOP, HDOP, VDOP). Red blocks indicate a problem even in perfect viewing conditions. Orange means a problem if there are obstructions below 10 degrees above of the horizon, yellow for less than 20 degrees ('urban canyon'). These calculations are very provisional and might be wrong!
